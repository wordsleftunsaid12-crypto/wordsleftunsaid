---
description: Git commit message conventions
globs: []
---

- Format: `type(scope): description`
- Types: feat, fix, refactor, test, docs, chore, style
- Scopes: shared, content, website, social, analytics, infra
- Examples:
  - `feat(content): add emotional template for message videos`
  - `fix(shared): handle null email in message type`
  - `chore(infra): add supabase local dev config`
- Keep subject under 72 characters
